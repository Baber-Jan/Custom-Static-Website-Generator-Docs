## Dependency

### Install and run IPFS
https://docs.ipfs.io/guides/guides/install/

### Run IPFS
{{< code lang="bash" >}}
ipfs init
ipfs daemon
{{< /code >}}

### Install Node js
https://nodejs.org/en/download/

## Run repo
{{< code lang="bash" >}}
npm install
npm start
{{< /code >}}

## Installing Whatsapp-web.js
Open a new terminal and get to the current directory.

### Run repo
{{< code lang="bash" >}}
npm install
npm start <http://<ip address>:<port no>/>
{{< /code >}}
